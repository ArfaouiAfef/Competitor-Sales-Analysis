 ## 📊 Project Overview

This project analyzes **sales performance and competitor behavior** across multiple countries using:

- 📊 **Power BI** for interactive dashboards  
- 🤖 **Machine Learning** for revenue prediction  

🎯 The goal is to transform raw data into **actionable insights** and support better decision-making.

---

## 🌍 Dataset

The dataset includes:

- **Sales** → Transactions (Revenue, Units, Date, Country)  
- **Products** → Product details (Category, Segment, Manufacturer)  
- **Geography** → Location data (Country, Region, City)  

📍 Countries analyzed:
- Canada 🇨🇦  
- Germany 🇩🇪  
- Japan 🇯🇵  
- Mexico 🇲🇽  
- Nigeria 🇳🇬

## 🤖 Machine Learning

### 🎯 Objective
Predict **Revenue** using historical data.

### 🧠 Models Tested
- Linear Regression  
- Random Forest  
- XGBoost ✅  

### 📈 Results
| Model             | Performance|
|-------------------|------------|
| Linear Regression | Medium     |
| Random Forest     | Good       |
| XGBoost           |⭐Best      |

👉 XGBoost provided the best results thanks to its ability to handle complex patterns.
